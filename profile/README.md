# ecotel-it

The engineering organization of Ecotel — ICT Excellence Center for the
planning, development, and management of mission-critical technology
infrastructure.

## Structure

- **[`.github`](https://github.com/ecotel-it/.github)** — org-wide CI workflows, policies, community files
- **[`branding`](https://github.com/ecotel-it/branding)** — visual assets and content design system
- **[`guidelines`](https://github.com/ecotel-it/guidelines)** — engineering standards (start with [REPO-NAMING.md](https://github.com/ecotel-it/guidelines/blob/main/REPO-NAMING.md))
- **`template-*`** — "Use this template" scaffolding for new projects *(coming in Phase 2)*
- **`<bu>-*`** — product and service repos, prefixed by business unit

## Business units

| Prefix     | Unit                                          |
|------------|-----------------------------------------------|
| `infra-`   | Infrastrutture e cablaggio                    |
| `netsec-`  | Networking, sicurezza e cyber security        |
| `products-`| Prodotti                                      |
| `apps-`    | Sviluppo applicativo                          |

Starting a new project? Read the [naming convention](https://github.com/ecotel-it/guidelines/blob/main/REPO-NAMING.md) first.

## Reporting a vulnerability

Please don't open public issues for security vulnerabilities. See
[SECURITY.md](https://github.com/ecotel-it/.github/blob/main/SECURITY.md).
