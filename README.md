# ecotel-it / .github

Org-wide community health files, policies, and (in Phase 2) reusable CI
workflows for the `ecotel-it` GitHub organization.

## What lives here today

- `profile/README.md` — rendered at [github.com/ecotel-it](https://github.com/ecotel-it)
- `SECURITY.md` — vulnerability reporting policy
- `CODE_OF_CONDUCT.md` — community standards (Contributor Covenant 2.1)
- `ISSUE_TEMPLATE/` — default issue templates for repos that don't override them

## What will land in Phase 2

- `workflows/` — reusable workflows (`lint.yml`, `test.yml`, `security-scan.yml`, `release.yml`) consumed by every repo via `workflow_call`.

## Related repos

- [`ecotel-it/guidelines`](https://github.com/ecotel-it/guidelines) — engineering standards (start with [REPO-NAMING.md](https://github.com/ecotel-it/guidelines/blob/main/REPO-NAMING.md))
- [`ecotel-it/branding`](https://github.com/ecotel-it/branding) — design system and brand assets
